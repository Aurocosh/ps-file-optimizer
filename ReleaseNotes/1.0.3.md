# 1.0.3

Stricter defaults when plugins are missing.

## Behavior

- Missing portable plugin bundle (no `Install-FoPlugins` tree) is a **hard error** with install instructions, unless `PluginSearchMode` is `PathOnly`.
- `MissingToolsPolicy` defaults to **`Error`**: any required tool that cannot be resolved fails the file.
- Opt-outs: `SkipTool` (skip steps that need missing tools) or `SkipFile` (skip the whole file).
- Legacy config/param `SkipMissingTools` still maps when present (`true` → `SkipFile`, `false` → `Error`).
