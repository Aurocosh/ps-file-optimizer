# 1.0.4

Fix `Install-FoPlugins -Mode FullPortable` omitting TruePNG and PNGOut.

## Fixes

- Plugin install inventory no longer uses shared `.ico` as the PNG sample extension (`.ico` maps to both ICO and PNG), which skipped TruePNG/PNGOut steps and left `truepng.exe` / `pngout.exe` out of the FullPortable copy plan.
- Inventory prefers a group-unique extension (for PNG, `.png`) so required PNG tools are installed.

## Notes

- Re-run `Install-FoPlugins -Mode FullPortable -Force` after upgrading so the missing EXEs are copied into `Plugins64` / `Plugins32`.
- Legacy `rehuff.exe` remains unused by pipelines (`rehuff_theora.exe` is installed instead).
